# 💪 今天练什么 (Workout Helper)

[中文](#中文) | [English](#english)

---

## 中文

这是一个轻量级、完全去中心化的静态网页应用，旨在帮助健身爱好者快速决定当天的训练内容，并记录训练进展。

### 🌟 项目目的

本项目致力于解决“健身选择困难症”。通过用户自定义的训练部位和工具组合，提供随机抽签推荐、专属视频跟练以及结构化的数据统计，让健身变得简单、有趣且高度个性化。所有数据均保存在您的浏览器本地（LocalStorage），不依赖任何服务器，保护隐私且安全。

### ✨ 核心功能

-   **🏗️ 高度自定义配置**：
    -   自由添加/删除训练部位（如：肩背、核心等）。
    -   自由添加/删除训练工具（如：哑铃、弹力带等）。
    -   **联动系统**：为每个部位配置适用的工具，确保抽签结果科学合理。
-   **🎲 智能推荐/抽签**：
    -   一键决定今天练哪个部位、用什么工具。
    -   随机展示运动小贴士，涵盖热身、技术及心理建设。
-   **📺 专属视频库**：
    -   支持粘贴 B 站视频链接，自动抓取标题。
    -   点击即可跳转跟练，构建您最熟悉的动作库。
-   **📅 训练记录与打卡**：
    -   记录训练时长、所用器械及心得备注。
-   **📊 数据统计对比**：
    -   对比近 3 天与近 7 天的训练数据。
    -   可视化展示训练分布，防止偏科。
-   **💾 配置备份**：
    -   支持导出/导入 JSON 配置，数据永不丢失。

### 🚀 使用方法

1.  **运行**：双击 `workout.html` 即可运行，或部署至 GitHub Pages。
2.  **配置**：进入“**我的配置**”设定部位、工具及其联动关系。
3.  **训练**：进入“**今日训练**”点击 🎲 按钮抽取计划。
4.  **打卡**：完成后填写时长并点击“**完成打卡**”。

## English

A lightweight, fully decentralized static web application designed to help fitness enthusiasts quickly decide their daily routines and track their progress.

### 🌟 Purpose

This project is built to solve "workout decision fatigue." By allowing users to customize muscle groups and equipment combinations, it provides random recommendations, personalized video libraries, and structured statistics. All data is saved locally in your browser (**LocalStorage**), ensuring 100% privacy and no server dependency.

### ✨ Key Features

-   **🏗️ High Customization**:
    -   Add or remove muscle groups and training tools freely.
    -   **Linkage System**: Configure which equipment goes with which muscle group for logical recommendations.
-   **🎲 Smart Recommendation**:
    -   One-click to decide "What to train" and "What to use."
    -   Random fitness tips covering warm-up, techniques, and mindset.
-   **📺 Exclusive Video Library**:
    -   Paste Bilibili links to automatically fetch titles.
    -   Click to open and follow along with your favorite routines.
-   **📅 Progress Tracking**:
    -   Log workout duration, equipment used, and personal notes.
-   **📊 Data Analytics**:
    -   Compare sessions and duration between the last 3 days and 7 days.
    -   Visual distribution bars to prevent training imbalances.
-   **💾 Configuration Backup**:
    -   Export and import your entire setup as a JSON file.

### 🚀 How to Use

1.  **Launch**: Double-click `workout.html` to run in any browser, or host it on GitHub Pages.
2.  **Setup**: Go to the "**My Config**" tab to define your muscles, equipment, and videos.
3.  **Train**: Go to "**Today's Training**" and click the 🎲 buttons to roll your plan.
4.  **Log**: After training, enter your duration and click "**Check In**."
